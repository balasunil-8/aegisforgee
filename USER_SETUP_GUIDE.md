## ğŸ› Troubleshooting

### Common Issues & Solutions

#### Issue 1: Python not found
```bash
# Install Python 3.8+ from python.org
# Make sure to check "Add Python to PATH"
python --version  # Verify installation
```

#### Issue 2: Port already in use
```bash
# Windows
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Linux/Mac
lsof -ti:5000 | xargs kill -9
```

#### Issue 3: Dependencies fail to install
```bash
# Create virtual environment first
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# Then install
pip install -r requirements.txt
```

#### Issue 4: Database not initializing
```bash
# Re-initialize databases
scripts/windows/init_databases.bat  # Windows
./scripts/linux/init_databases.sh   # Linux/Mac
```

**For more help**: See `docs/installation/troubleshooting.md`
